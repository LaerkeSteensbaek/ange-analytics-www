Case: TAPLA - The Tanker Planner

In 2016 we started the development of TAPLA, the tanker planner - a software tool to assist expert users in making the optimal tanker plan when coordinating air-to-air refuelings.

The tool was developed in close collaboration with the SMEs, iteratively capturing their knowledge in advanced mathematical models, visualizing the scenarios, incorporating feedback and iterating...

Features

Reduce cost

  • Minimize the tanker consumptions
  • Reduce the average number of tankers needed to support an ATO
  • Optimize the allocation of tanker types to receiver types based on
    • off-load requirements
    • compatibilities
    • flight times

Increase efficiency

  • Reduces the time to make a plan from several hours to minutes
  • Automatically satisfy technical constraints (boom / drogue / soft basket compatibility / flight time)
  • Automatically satisfy operational constraints
  • Increases the SA of the planner

Integrate

  • Can be used as a stand-alone product or integrated into the existing IT infrastructure
  • Facilitates the integration of the planning process with adjacent functions
  • Built using best practice & state-of-the art software development principles
  • Enables automated performance monitoring and Data-Driven Decisions
